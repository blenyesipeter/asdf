Miért jó a GitHub?

Nem csak kódtemető. Meglepő, tudom.

Kattints egy random gondolathoz.

Verziókezelés

A GitHub megőrzi a módosításokat, így vissza lehet menni korábbi verziókra ha valamit elrontasz. Amit úgyis el fogsz.

Csapatmunka

Többen dolgozhatnak ugyanazon projekten pull requestekkel, branchekkel és merge-ekkel.

Portfólió

Publikus repókban megmutathatod a projektjeidet. Gyakorlatilag digitális bizonyíték, hogy tudsz valamit.

GitHub Pages

Ingyen hosztolhatsz statikus weboldalakat. Ez az oldal is mehet oda.

Backup

Nem csak a gépeden van a projekt. Ha meghal a laptop, kisebb az egzisztenciális válság.

Open Source

Mások kódját tanulhatod, hozzájárulhatsz projektekhez, és láthatod hogyan dolgoznak profik.

GitHub: mert az "asztalon hagyott random mappa" nem fejlesztési stratégia.
